# AI Agent Identity Presentation

A comprehensive web-based presentation exploring the fundamentals of identity and access management for AI agents and autonomous intelligent systems.

## Overview

This presentation addresses the critical question of **trust in autonomous intelligent systems** and provides a framework for securing AI agents through the **Four A's of Agent Identity**: Administer, Authenticate, Authorize, and Audit.

## Getting Started

### Running Locally

1. Clone this repository or download the zip file and extract it
```bash
git clone https://github.com/yourusername/ai-agent-identity-presentation.git
cd ai-agent-identity-presentation
```

2. Open `index.html` in your web browser:
```bash
open index.html  # macOS
# or
start index.html  # Windows
# or
xdg-open index.html  # Linux
```

Alternatively, use a local server:
```bash
python3 -m http.server 8000
# Then navigate to http://localhost:8000
```

### Navigation

- **Next**: Right Arrow, Space, or Enter
- **Previous**: Left Arrow
- **Slide Counter**: Bottom navigation shows current slide number
